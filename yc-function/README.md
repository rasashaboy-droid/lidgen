# Лидген — функция отправки заявок на почту

Принимает POST с данными квиза, шлёт письмо на `chistoleads@mail.ru` через SMTP mail.ru.

## Деплой в Yandex Cloud (пошагово)

### 1. Создать функцию
1. Зайти в [console.cloud.yandex.ru](https://console.cloud.yandex.ru) → Cloud Functions → **Создать функцию**
2. Имя: `lidgen-mail`
3. Создать → **Создать в редакторе**

### 2. Загрузить код
В редакторе:
- Среда выполнения: **nodejs18** (или новее)
- Точка входа: `index.handler`
- Таймаут: **10 секунд**
- Память: **128 MB**

Файлы:
- `index.js` — код функции (из этой папки)
- `package.json` — зависимости

Можно либо скопировать содержимое файлов в редактор, либо загрузить ZIP-архив папки `yc-function/`.

### 3. Переменные окружения
Добавить в разделе «Параметры» → «Переменные окружения»:

| Ключ | Значение |
|---|---|
| `SMTP_HOST` | `smtp.mail.ru` |
| `SMTP_PORT` | `465` |
| `SMTP_USER` | `chistoleads@mail.ru` |
| `SMTP_PASS` | *пароль приложения mail.ru (16 символов)* |
| `MAIL_TO` | `chistoleads@mail.ru` |
| `ALLOWED_ORIGINS` | `https://rasashaboy-droid.github.io` (через запятую можно добавить ещё — `http://localhost:3456` для тестов, новый домен когда появится) |

### 4. Создать версию и опубликовать
Нажать **Создать версию**.

### 5. Сделать функцию публичной
В функции → вкладка «Обзор» → раздел «Доступ к функции»:
- Включить **«Публичная функция»** (любой может вызывать без авторизации)

### 6. Скопировать URL
В разделе «Обзор» будет ссылка вида:
```
https://functions.yandexcloud.net/d4xxxxxxxxxxxxxxx
```
Это и есть URL функции — отправь его, чтобы подключить к сайту.

## Тест вручную (curl)
```bash
curl -X POST https://functions.yandexcloud.net/ВАШ_ID \
  -H "Content-Type: application/json" \
  -d '{"phone":"+79991234567","q1":"15-20","q2":"10-15","q3":"Telegram"}'
```
Ожидаемый ответ: `{"ok":true}` + письмо приходит на почту.
